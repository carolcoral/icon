# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-09-23

### 新增功能
- 🎉 **初始版本发布** - 完整的在线图标库系统
- ✨ **多格式图片支持** - 支持PNG、ICO、JPG、JPEG、GIF、SVG格式
- 🗂️ **智能分类管理** - 自动根据文件夹结构创建分类
- 🔍 **实时搜索功能** - 支持关键词搜索过滤
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 👁️ **图片预览功能** - 点击图片查看大图预览
- 📥 **下载功能** - 支持图片下载
- 🔗 **复制链接** - 快速获取图片直链

### 技术特性
- **前端框架**: Vue 3.3.4 + Composition API
- **构建工具**: Vite 4.4.5
- **样式框架**: Tailwind CSS 3.3.0
- **UI组件**: Headless UI + Heroicons
- **架构**: 纯静态部署，无后端依赖

### 部署支持
- ✅ Vercel
- ✅ Netlify  
- ✅ GitHub Pages
- ✅ Nginx
- ✅ EdgeOne Pages
